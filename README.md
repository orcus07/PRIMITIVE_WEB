# 회의록 작성기 (Meeting Minutes)

브라우저에서 바로 동작하는 간단한 회의록 작성 웹앱입니다. 별도의 서버나
설치가 필요 없으며, 작성한 회의록은 브라우저의 `localStorage`에 저장됩니다.

## 기능

- 📝 회의록 작성 — 제목, 날짜, 참석자, 안건, 논의 내용, 결정 사항, 실행 항목
- 💾 자동 저장 — 브라우저 localStorage에 보관 (서버 불필요)
- 📚 목록 관리 — 저장된 회의록 목록을 날짜순으로 확인
- 🔍 검색 — 제목·내용으로 빠르게 찾기
- 🖨️ 인쇄 / PDF — 인쇄 친화적 레이아웃으로 출력
- 📄 텍스트 내보내기 — `.txt` 파일로 다운로드

## 사용 방법

`index.html` 파일을 브라우저로 열기만 하면 됩니다.

```
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
```

## 파일 구성

| 파일 | 설명 |
| --- | --- |
| `index.html` | 화면 구조 (마크업) |
| `style.css` | 스타일 및 인쇄용 레이아웃 |
| `app.js` | 회의록 저장·검색·내보내기 로직 |

## 데이터 보관

데이터는 사용 중인 브라우저에만 저장됩니다. 다른 기기·브라우저와는 공유되지
않으며, 브라우저 데이터를 지우면 회의록도 함께 삭제되니 중요한 내용은
"텍스트 내보내기"로 백업해 두세요.
