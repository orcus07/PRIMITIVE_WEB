# 📰 영문 링크 → 한글 증류 리더

영어 장문 자료의 링크를 던지면, **원문 의도에 충실하게 한글로 번역·구조화하고 핵심을 증류**해
바로 읽을 수 있는 자료로 만들어 줍니다. 독자 페르소나는 **SK하이닉스 반도체 마케터(AI 동향 관심)** 로
맞춰져 있어, 요약·인사이트가 그 관점에서 정리됩니다.

## 어떻게 동작하나

```
영문 링크 → 본문 수집(3중 전략) → Claude가 한글로 번역·구조화·증류 → 읽을거리 + 보관함 저장
```

**본문 수집 전략** (봇 차단 사이트 대응):
1. 브라우저처럼 직접 요청
2. 막히면 리더 프록시(`r.jina.ai`) 우회
3. 그래도 막히면 → 화면에서 **본문 직접 붙여넣기**

**결과물** (반도체 마케터 · AI 동향 관점):
- 한 줄 핵심 (증류)
- 이 글의 주제·맥락
- 📌 마케터 인사이트 (왜 읽을 가치가 있는지)
- 원문 흐름을 따른 충실한 한글 본문 정리 (요약 아님)
- 핵심 용어·숫자 풀이
- → 보관함에 저장되고 검색 가능

요약·번역 엔진: **Claude `claude-opus-4-8`**

## 준비물

1. **Node.js 20 이상** — https://nodejs.org
2. **API 키 1개**: `ANTHROPIC_API_KEY` — https://console.anthropic.com

> OpenAI 키는 필요 없습니다(이전 버전과 달라진 점).

## 실행 방법

```bash
npm install
cp .env.example .env      # ANTHROPIC_API_KEY 입력
npm start                 # http://localhost:3000
```

## 사용법

1. 상단에 영문 아티클 링크를 붙여넣고 **증류**(또는 Enter)
2. 잠시 후 한글 정리 결과가 표시되고 왼쪽 보관함에 저장됨
3. 링크가 봇 차단으로 막히면 → "본문 직접 붙여넣기"로 원문을 붙여넣어 처리
4. 보관함에서 검색하거나 지난 글을 다시 열어볼 수 있음

## 파일 구성

```
src/server.js           서버 (링크 → 수집 → 증류)
src/lib/fetchArticle.js 본문 수집 (직접 요청 → 프록시 우회)
src/lib/distill.js      Claude 번역·구조화·증류 (페르소나 반영)
public/                 화면(UI) + 보관함
```

## 참고 / 한계

- 일부 사이트(예: Cloudflare 보호)는 직접/프록시 모두 막힐 수 있어요 → 본문 붙여넣기로 우회
- 매우 긴 글은 출력 한도(16K 토큰)로 일부 잘릴 수 있습니다
- 보관함 데이터는 사용하는 브라우저(localStorage)에만 저장됩니다
